How to run it

The program is written in pycharm, but it can be run in any program that supports python.

I made a python "Eth-tracker" app that gives you 4 options:

1. In the console input, enter any address of the wallet you are interested in and the program prints the current amount of Etherium on it.

2.View the transactions that are connected to the wallet that you have entered in a human readable way.

3. View all transactions on the time chart.

4. View transactions from the start block you enter to the last block entered.

After running the program you shoud enter the number of the option you want
